# Outlook Ticket Automation Dashboard

## 📌 Project Overview
This project automates ticket tracking from Outlook email alerts using Power Automate, SharePoint, and Power BI.

## ❗ Business Problem
Ticket alerts arrive via email (ServiceNow, Fieldglass) and are unstructured. Manual tracking leads to inefficiencies and lack of visibility into SLA performance.

## 💡 Solution
Built an automated pipeline:

Outlook → Power Automate → SharePoint → Power BI

## 🔧 Key Features
- Automated email parsing
- Ticket categorization & prioritization
- SLA tracking and aging analysis
- Interactive Power BI dashboard
